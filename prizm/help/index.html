B) Font Used

You can change font from CSS file.

C) Color Schema

Prizm comes with 6 predefined colors green, blue, orange, purple, yellow and grey.

You can change the color of theme by the following code in demo HTML sources, <head> tag: 

<link class="colors" rel="stylesheet" href="layout/colors/green.css" type="text/css">

D) Countdown Date

To change the date of the countdown, go to layout/js/ folder and open main.js file. Find to:

date:"Dec 25, 2017 20:30:00"

and change to the date you want.

E) Background Image - Video

Single Image

Open image.html in a text editor. You'll see the following code:

$("header").backstretch("images/slide/1.jpg");

Image Slideshow

Open slide.html in a text editor. You'll see the following code:

$("header").backstretch([
	"images/slide/1.jpg", 
	"images/slide/2.jpg", 
	"images/slide/3.jpg", 
	"images/slide/4.jpg"
], {duration:3000, fade:750});

Youtube Background

Open video.html in a text editor. You'll see the following code:

<div id="BackgroundVideo" class="player" 
    data-property="{videoURL:'https://www.youtube.com/watch?v=t4CUNVYxxZM', containment:'header', autoPlay:true, showControls:false, mute:false, startAt:0, opacity:1, showYTLogo:false, vol:25}">
</div>

You can the video by replacing t4CUNVYxxZM with your own Youtube video ID.

F) MailChimp

MailChimp is intergrated to Prizm.

Mailchimp (subscription) in the file changes main.js.

In url parameter you have to put your signup form URL. Follow these steps to get this URL address:

  1. Login to MailChimp and go to Lists
  2. Open your chosen list and click on Signup forms
  3. Click on General forms
  4. Find Signup form URL, copy the URL
  5. Open in your browser this page: www.unshorten.it, paste Signup form URL and press unshorten button
  6. You'll resolve URL similar to this one: 
http://xxx.xxx.list-manage.com/subscribe?u=xxx&id=xxx

Please add /post after subscribe:

http://xxx.xxx.list-manage.com/subscribe/post?u=xxx&id=xxx

Finally, copy URL and paste to url in main.js

http://xxx.xxx.list-manage.com/subscribe/post?u=xxx&id=xxx

G) Contact Form

You need to configure contact.php to get e-mails from the users.

Change those variables on line 11 with the real name & e-mail address to send the mail information:

$toName = "Prizm";
$toAddress = "email@sitename.com";
